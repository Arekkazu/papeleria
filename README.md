# Papelería Papeleria - Proyecto Frontend

## Descripción

**Papelería Papeleria** es una aplicación frontend desarrollada en **React** que simula un sitio web de una papelería. Actualmente, el proyecto se encuentra en la etapa de diseño, sin funcionalidad backend ni lógica de roles.

Este proyecto está enfocado en el desarrollo frontend, por lo que se centra en la creación de la estructura y el diseño visual. La página incluye secciones de productos, carrito de compras y detalles de contacto.

## Tecnologías Utilizadas

- **React**: Librería JavaScript para construir la interfaz de usuario.
- **CSS**: Para el estilo y diseño visual de la aplicación.

## Instalación

1. Clona este repositorio a tu máquina local:

   ```bash
   git clone https://github.com/tu-usuario/papeleria.git
   ```

2. Navega al directorio del proyecto:

   ```bash
   cd papeleria
   ```

3. Instala las dependencias necesarias:

   ```bash
   npm install
   ```

4. Ejecuta la aplicación en modo de desarrollo:

   ```bash
   npm start
   ```

   Esto abrirá la aplicación en tu navegador, generalmente en `http://localhost:3000`.

## Estructura del Proyecto

- **public/**: Archivos estáticos como `index.html` y las imágenes.
- **src/**: Código fuente de la aplicación.
  - **components/**: Componentes reutilizables.
  - **pages/**: Páginas principales de la aplicación (por ejemplo, página de inicio, productos).
  - **assets/**: Archivos estáticos, como imágenes y estilos.
  - **App.js**: Componente principal de la aplicación que contiene la lógica de enrutamiento.

## Funcionalidades Actuales

- **Interfaz de usuario**: El diseño está inspirado en diferentes paginas de papelerias colombianas.
- **Páginas principales**:
  - Página de inicio
  - Página de productos
  - Página de carrito de compras
  - Página de contacto
